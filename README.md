# Personal Portfolio V2 🚀

Not a complete one but it's my another portfolio and I want to share with y'all. This version I made is based on Mr.JackieZhang portfolio with interactive UI and dragable elements. Please note that I have stopped updating this project for a while so some of the UI might be not as expected including my old resume.


## 🌟 Features

- **Modern UI/UX**: A clean and responsive design built with Tailwind CSS.
- **Dynamic Animations**: Seamless transitions and animations powered by Framer Motion and GSAP.
- **Physics-Based Elements**: Interactive physics simulations using Matter.js to make the experience more fun and engaging.
- **Project Showcase**: A detailed section presenting my latest and greatest projects, complete with links, descriptions, and technology stacks.
- **Interactive Timeline**: A visual journey through my professional experience and education.
- **Tech Stack Overview**: A visual representation of the programming languages, frameworks, and tools I use.
- **Contact Form**: An easy way for visitors to reach out directly from the portfolio.

## 🛠️ Technology Stack

This project is built using modern web development tools to ensure high performance and great developer experience:

- **Frontend Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**:
  - [Framer Motion](https://www.framer.com/motion/)
  - [GSAP](https://gsap.com/)
- **Physics Engine**: [Matter.js](https://brm.io/matter-js/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Utilities**: `clsx`, `tailwind-merge`, `class-variance-authority`

## 📁 Project Structure

Here is a quick overview of the main sections in the app (located in `src/components/`):

- `NavBar`: Main navigation header.
- `Hero`: The welcoming banner and introduction.
- `AboutMe`: A section diving into who I am and my background.
- `TechStackV2`: The current interactive tech stack display.
- `Project`: A showcase of my featured works.
- `Timeline`: My professional and educational roadmap.
- `ContactMe`: A functional contact form for inquiries.
